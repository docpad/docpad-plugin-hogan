# Hogan Plugin for DocPad
Adds support for the [Hogan](http://twitter.github.com/hogan.js/) templating engine to [DocPad](https://docpad.org)

Convention:  `.anything.hogan`


## Install

```
npm install --save docpad-plugin-handlebars
```
